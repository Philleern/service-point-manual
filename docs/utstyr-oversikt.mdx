# Utstyr - oversikt

PDF-en beskriver følgende som standardutstyr for pakkeutlevering:

1. Håndterminal for ut- og innlevering av pakker
2. Etikettprinter for utskrift av etiketter
3. Hyllekoder som skannes når pakker legges på plass
4. Profilering i form av skilt for vindu og/eller skap

## Slik brukes oversikten

- Se [Håndterminal](./handterminal) for daglig bruk og oppstart.
- Se [Etikettprinter](./etikettprinter) for utskrift og feilsøking.
- Se [Hyllekoder og profilering](./hyllekoder-og-profilering) for fysisk oppsett.

## Kontrollpunkt

- Alle hyllekoder må være skannbare.
- Printer og PDA må være klare før første kunde.
- Profilering må være synlig for kundene.
