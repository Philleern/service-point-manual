# Ankomst sjåfør

## Når brukes dette?

Når sjåføren leverer inn pakker som skal registreres og plasseres i hyller for senere henting.

## Mål

Hver pakke skal få riktig hyllelokasjon slik at den blir enkel å finne igjen ved utlevering.

## Kontrollpunkt

- Både lokasjon og kolli-ID må være skannet før pakken er registrert ankommet.
- Nyeste skann på kolliet blir gjeldende lokasjon. Hvis samme pakke først skannes på `A1` og senere på `A2`, blir `A2` stående.

:::tip Tips
Spre pakkene over flest mulig hyllelokasjoner. Det gjør dem raskere å finne ved henting.
:::

:::note NB
PDF-en anbefaler å skanne esker først og poser etterpå. Det gjør sorteringen i hyllene enklere.
:::

## Vanlige feil

- Feilmelding ved ankomstregistrering: følg teksten i PDA og kontroller at riktig etikett er skannet.
- Fysisk pakke uten lokasjon i systemet: se [Mangler ankomstregistrering](./mangler-ankomstregistrering).
