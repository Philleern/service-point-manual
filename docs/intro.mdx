# PostNord Service Point / Mypack Collect-håndbok

Denne håndboken er laget for ansatte som bruker PDA/håndterminal til å motta, plassere, utlevere og returnere pakker i PostNord Service Point.

## Hva er dette?

- En oppgavebasert manual for daglig drift.
- En praktisk oversettelse av innholdet i PDF-en `MypackHåndbok V1.1`.
- Et docs-as-code-oppsett som kan oppdateres fortløpende med nye rutiner og screenshots.

## Målgruppe

- Butikkmedarbeidere og vikarer som jobber med utlevering og retur.
- Ansvarlige for oppstart av nye utleveringssteder.
- Drift og support som trenger en felles referanse.

## Slik bruker du manualen

- Start i riktig oppgave i sidebaren, ikke i gammel kapittelrekkefølge.
- Følg nummererte steg og stopp ved hvert kontrollpunkt.
- Bruk feilsidene når PDA eller printer ikke oppfører seg som forventet.
- Legg inn faktiske screenshots fortløpende ved å følge [Screenshots-guide](./screenshots-guide).

:::tip Tips
Nøkkelord i denne håndboken følger ordlyden i PDA-menyene, for eksempel `Ankomst sjåfør`, `Utlevering`, `Sende pakke / Innlevert retur` og `Uavhentet retur`.
:::

## Hva inneholder manualen?

- Oppstart av arbeidsdagen
- Mottak og plassering av pakker
- Ordinær utlevering og utlevering med ID-kontroll
- Returflyt og manglende ankomstregistrering
- Utstyr, support og endringslogg

## Kontakt

Se [Kontakt og support](./kontakt) for telefonnumre og anbefalt eskalering.

:::warning Viktig
PDF-en som denne manualen bygger på oppgir versjon `V1.1`, men inneholder begrenset detaljert endringshistorikk. Se [Endringslogg](./endringslogg) for hva som er kjent og hva som må fylles inn senere.
:::
