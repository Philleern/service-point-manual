# Hyllekoder og profilering

## Hyllekoder

Hyllekoder er strekkoder som festes på hyllene og brukes når pakker skannes inn med `Ankomst sjåfør`.

## Anbefalt praksis

- Bruk tydelige koder som `A1`, `A2`, `B1`, `B2`.
- Sørg for at koden er enkel å skanne uten at pakken må flyttes først.
- Spre pakkene over flere hyller for raskere gjenfinning.

## Profilering

PDF-en beskriver profilering som skilt i vindu og/eller skap. Dette settes normalt opp av distriktleder.

## Kontrollpunkt

- Hyllekoder må være lesbare og fysisk på plass.
- Hvis nyeste skann skal gjelde, må den faktiske pakkeplasseringen samsvare med siste lokasjon som ble skannet.
