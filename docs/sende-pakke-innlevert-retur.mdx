# Sende pakke / Innlevert retur

## Når brukes dette?

Når kunden leverer inn en pakke med ferdig etikett som skal sendes videre eller registreres som retur.

## Mål

Registrere innleveringen korrekt, sende kvittering og plassere pakken i riktig returflyt.

## Kontrollpunkt

- Bruk denne funksjonen bare når pakken allerede har ferdig etikett.
- Pakken må være forsvarlig emballert før du registrerer den inn.
- Kvittering bør sendes før pakken legges fra deg.

## Vanlige feil

- Kunden har bare referanse og trenger etikett først: bruk [Print etikett](./print-etikett).
- Kvittering kan ikke sendes: kontroller at minst ett kontaktfelt er fylt inn.
- Printeren svarer ikke i etikettflyten: se [Printer-problemer](./printer-problemer).
