# Utlevering med etterkrav

## Status

PDF-en `MypackHåndbok V1.1` nevner `Utlevering med etterkrav` i innholdsfortegnelsen, men inneholder ikke en komplett prosedyreside for denne flyten.

## Hva betyr det i praksis?

- Denne docs-siden er opprettet for å holde plass i informasjonsarkitekturen.
- Rutinen må bekreftes mot gjeldende driftsprosess før siden fylles ut.
- Ikke anta at ordinær utlevering kan brukes uendret når betaling kreves.

:::warning Viktig
Behandle denne siden som `Ikke i bruk / Ikke dokumentert` til PostNord har bekreftet gjeldende arbeidsflyt for etterkrav.
:::

## Neste steg

1. Skaff oppdatert prosess fra drift eller support.
2. Legg inn faktisk PDA-flyt og betalingskontroll.
3. Oppdater denne siden og [Endringslogg](./endringslogg).
